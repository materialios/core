# @materialios/core
ReactJs core for @materialios

## Features
* MaterialiosProvider

## Installation
```sh
npm i @materialios/core
```

## Example
```c
import { MaterialiosProvider } from '@materialios/core'

const App = () => {
  return (
    <div className='App'>
      <MaterialiosProvider>
        <Home />
      </MaterialiosProvider>
    </div>
  )
}

export default App
```
